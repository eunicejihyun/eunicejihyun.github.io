competitive dance studio

pythonflasksqlalchemy

A dance studio needed to streamline their request management process. Excel spreadsheets weren’t powerful enough to keep up with the complexity of national dance competition rules as well as their own studio’s decision making process.

To move this studio to an upgraded tech solution, I created a full-stack web application to allow them to keep track of and run reports on their data. This CRUD app is in use by more than a dozen of their staff.

In addition to saving time, the request management system equips the management team and staff of this studio with the ability to:

  • view QA errors and duplicate/competing requests via reports
  • edit, approve, reject, and view historical changes to a request with ease
  • view details for each student and teacher
  • view updated statuses for relevant requests (only with certain permissions)
  • create custom invitation letters when used in combination with Office Word
  • enforce deadlines for request submission
